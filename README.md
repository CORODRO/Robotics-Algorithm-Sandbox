# Testing Codes
This repository contains:

Examples of preliminary ROS CODES:

1 - CV_bridge_demo.py --> translate the readings of a Depth Camera in color-DEM-map

2 - Fake_localization.py --> Generate a fake localizator to only test the path planning. The code publish a fake reading of a position.

3 - Fake_mapper.py --> Generate a fake mapper to test only the path planning.The code publish a fake reading of a gridmaps.

4- First_teleop_code_implemented_in_python --> very first code to control the turtlebot implemented on Python.

5 - Create_map_ROS.py --> translate LiDAR readings in MAPS

6 - laserscan_from_git_used_to_create_ROS_Map.py --> used in the Create_map_ROS.py

7 - ros2opencv2.py --> Wrapper between OpenCV (SLAM) and ROS

Examples of simple AI algorithms --> AI General

Examples of a better implemented DWA --> DWA Recursive

Examples of simple path planning and SLAM algorithm not directly used --> Navigation and PathPlanning Codes not Used
