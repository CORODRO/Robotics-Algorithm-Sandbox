# AI General Examples

Small algorithm exercises that helped the team think through planning, routing, optimization, and reinforcement-learning ideas before integrating them into robotics code.

## Included Topics

- Q-learning on grid-like toy problems.
- Vehicle routing examples.
- Simple traveling salesperson variants.
- Knapsack examples.
- Small plotting and synthetic environment scripts.

## Best Use

Use this folder for concept learning and classroom discussion, not as drop-in mission software.
