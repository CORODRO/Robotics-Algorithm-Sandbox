# AI General Examples

Small algorithm exercises that helped the team think through planning, routing, optimization, and reinforcement-learning ideas before integrating them into robotics code.

## Included Topics

- Q-learning on grid-like toy problems.
- Vehicle routing examples.
- Simple traveling salesperson variants.
- Knapsack examples.
- Small plotting and synthetic environment scripts.

## Script Guide

- `simple_q_learning_demo.py`: the smallest Q-learning example in the folder.
- `mdp_q_learning_example.py`: introductory MDP-style Q-learning exercise.
- `greedy_tsp_vs_q_learning.py`: compares a greedy route heuristic with a learning-based approach.
- `simple_tsp_demo.py`: small travelling-salesperson toy problem.
- `vehicle_routing_q_learning.py`: Q-learning on a hand-built routing reward matrix.
- `vehicle_routing_with_graphs.py`: routing example with waypoint coordinates and optional graph plotting.
- `dynamic_programming_knapsack.py`: compact knapsack dynamic-programming example.
- `variable_weight_knapsack.py`: knapsack variant with changing/dependent weights.
- `networkx_graph_q_learning.py`: graph-based path-learning experiment.
- `empirical_moon_soil_rock_distribution.py`: synthetic terrain/rock-distribution exploration.
- `plotting_3d_experiment.py`: simple 3D plotting sandbox.
- `keras_pima_diabetes_example.py`: small supervised-learning example kept as a side reference.

## Best Use

Use this folder for concept learning and classroom discussion, not as drop-in mission software.

Most files here are standalone Python scripts and are the easiest place to start if you are new to the repository.
